# 🏥 Medical Clinic Management System

> نظام إدارة عيادة طبية متكامل

## 📋 الوصف
نظام سطح مكتب لإدارة العيادات الطبية يشمل ملفات المرضى،
المواعيد، السجلات الطبية، والفواتير.

## ⚙️ التقنيات المستخدمة
- **Delphi 12** — الواجهة والمنطق البرمجي
- **Microsoft Access** — قاعدة البيانات

## ✨ المميزات
- تسجيل المرضى وملفاتهم الطبية
- جدولة مواعيد الأطباء
- متابعة الوصفات والتشخيصات
- إدارة الفواتير والمدفوعات
